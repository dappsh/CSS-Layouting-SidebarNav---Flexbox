SIDEBAR NAVIGATION LAYOUT

Bagaimana saya membuat halaman ini?

09 Mei 2018 - CSS LAYOUT
Salah satu tugas pada materi ini ialah membuat Sidebar Navigation Layout, dimana tampilannya seperti teman-teman lihat saat ini. Saya membuat Sidebar Navigation Layout ini menggunakan property Float Left & Float Right. Float Left digunakan untuk membuat navigation pada sisi kiri & Float Right untuk membuat sisi kanan yang berisi content. Float Left dan Float Right hanya bisa digunakan untuk membagi halaman menjadi 2 layout (kanan dan kiri). Jika teman-teman ingin membagi halaman menjadi 3 atau lebih, sebaiknya gunakan cara lain.
Contohnya, pada tugas 3 kolom saya menggunakan columnn-count pada CSS

Float Left Content

  • Dibuat di dalam tag <div> dengan class "sidebar" untuk kebutuhan styling
  • Pada CSS class "sidebar" diberikan property float: left; dengan width="15%" dari halaman
  • Menggunakan tag <center> untuk membuat seluruh content berada di tengah
  • Background    : pada css class "sidebar" property background dengan warna turquoise
  • Click & Go    : dibuat dengan tag <h1> agar ukuran besar & bold
  • Link on Image   : dengan tag <a href="url"> berikan tag <img> untuk menambahkan logo, maka link akan terbuka pada saat user klik logo tersebut</a>
  • Ukurang logo di adjust menggunakan property width="dengan satuan px"

Float Right Content

  • Dibuat di dalam tag <div> dengan class "paragraf" untuk kebutuhan styling
  • Pada CSS class "paragraf" diberikan property float: right; dengan width="83%" dari halaman
  • Pada CSS class "paragraf" diberikan property text-align: justify > text rata kanan & kiri
  • Membuat beberapa paragraf mengenai bagaimana halaman ini dibuat dengan tag <div> & beberapa tag sederhana <h1> <h2> <i> <b> <br>
  • Untuk membuat list, saya menghilangkan default bullet dan menggantinya dengan bullet berwarna yang dibuat di dalam tag <style> / CSS
